# SCOPEY

Scopey is a tool that allows teachers to describe the complex multi-layered
hierarchies of jargon that are required for learning new languages. Initially it
was made just for use in web development:

![a gif of the scope explorer examining some JavaScript](docs/scopeoscope-2.gif?raw=true)

But it could also be used for learning people languages:

![an English sentence being examined](docs/scopeoscope-english.png?raw=true)
